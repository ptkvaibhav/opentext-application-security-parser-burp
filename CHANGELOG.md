# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-10

### Added
- Initial release of the Burp Suite parser for Fortify SSC (OpenText Application Security).
- Parsing of Burp Suite XML exports into Fortify issues.
- Support for modern Fortify Plugin SPI (`1.2.2320.0`).
- Custom vulnerability attributes mapping for Confidence, Remediation Background, etc.
- Enterprise-grade CI/CD workflows, linting (Checkstyle), and security scans (OWASP Dependency-Check).
- Comprehensive documentation and GitHub repository templates.
